# Kerala Movies EDA

## Project Objective
Explore trends and insights in Kerala movie data, focusing on release years, genres, and ratings to understand factors behind the recent success of Kerala films.

## Folder Structure
- `data/`: Raw and cleaned datasets
- `notebooks/`: Jupyter notebooks for analysis and visualization
- `scripts/`: Python scripts for data preprocessing
- `images/`: Charts and figures
- `requirements.txt`: Project dependencies
